# 마이데이터 운영

## 운영 대상 및 주요 사항

```
정보주체
  ↓ 개인정보 전송요구
정보 전송자(본인, 제 3자_의료/공공기관)
  ↓ 보건의료정보 제공
중계전문기관 (의정원)
  ↓ 전송 중계 및 이력관리
특수전문기관
  ↓ 정보 수신·관리·활용
국민 대상 의료 마이데이터 서비스
```
